PRD · 定价条件单

需求说明
1、股票展示区域
  • 展示股票当前信息：名称、代码、当前价、涨跌幅（不自动刷新）。
  • 点击股票代码右侧图标，跳转快捷交易页面，允许用户切换股票。
2、条件单参数填充区域
2.1 监控价
  • 触发自动下单的价格门槛，以 0.0001 为步长增减，变动后计算并显示涨跌幅（涨红跌绿，为 0 隐藏）。
  • 涨跌幅公式：((监控价 - 昨收价)/昨收价)×100%，四舍五入到小数点后两位。
  • 当监控价涨跌幅 < 股价涨跌幅时，提示“当股价 ≤ 监控价时触发买入”；否则提示“当股价 ≥ 监控价触发买入”。
  • 需大于 0。
2.2 委托价
  • 触发后真正报给通道的价格，步长 0.0001。
  • 监控价变动后自动同步到委托价。
  • 需大于 0。
2.3 委托数量

触发后真正报给通道的数量，支持快捷选择；规则与【交易下单】一致。

2.4 截止时间

默认当日收盘前。

2.5 持仓与购买力

展示该 TRS 账户该标的的持仓数量与可用购买力。

3、功能区域
  • 确定下单：生成一条状态为“待触发”的条件单记录，并启动监听线程。
